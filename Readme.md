# ⚙️ Backend for MyApp

This is the backend of **MyApp**, built using Node.js, Express, and MongoDB. It includes user authentication with JWT and role-based access control for admin, seller, and customer.

---

## 📦 Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB + Mongoose**
- **JWT (JSON Web Tokens)**
- **bcrypt.js**
- **cookie-parser**
- **dotenv**

---

## 🧪 Features

- 🔐 User Authentication (Register/Login)
- 🎭 Role-Based Access (Admin, Seller, Customer)
- 🧾 Protected Routes with JWT (HttpOnly Cookies)
- 🧵 RESTful API structure (MVC pattern)
- 📦 MongoDB integration using Mongoose
- 📤 Multer for image uploads
- ☁️ Cloudinary for media storage
